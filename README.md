# Netflix UI Clone 🎬

This is a static Netflix homepage UI clone built using HTML and CSS.

## 🔧 Tech Stack
- HTML
- CSS

## ✨ Features
- Netflix-style homepage layout
- Banner section with overlay content
- Interactive buttons (Play & More Info)
- Horizontal scrollable carousel
- Hover effects on thumbnails

## 📌 Note
This is a static project focused on UI design. Responsiveness improvements are planned for future updates.
